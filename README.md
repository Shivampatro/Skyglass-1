# 🌤️ Skyglass — Atmospheric Intelligence

**Skyglass** is a high-performance, aesthetically-driven weather dashboard designed to provide real-time atmospheric data with a premium, glassmorphic interface. It leverages the OpenWeatherMap API to deliver precise conditions for any location on Earth.

![Skyglass Preview](https://github.com/Shivampatro/Skyglass-1/raw/main/index.html) *(Replace this with a screenshot later!)*

## ✨ Features

- **Real-time Weather**: Current temperature, feels-like, and condition descriptions.
- **Detailed Atmos Stats**: Humidity, wind speed/direction, visibility, pressure, and cloud cover.
- **Solar Windows**: Sunrise and sunset times with a custom solar tracking UI.
- **Dynamic Backgrounds**: Animated "sky orbs" that create a deep, immersive atmosphere.
- **Glassmorphic UI**: High-end transparency and blur effects for a modern look.
- **Responsive Design**: Optimized for desktops, tablets, and phones.

## 🚀 Tech Stack

- **Frontend**: HTML5, Vanilla JavaScript, CSS3.
- **Typography**: Google Fonts (Syne, DM Mono).
- **API**: [OpenWeatherMap API](https://openweathermap.org/).
- **Design Style**: Glassmorphism / Atmospheric Dark Mode.

## 🛠️ Usage

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Shivampatro/Skyglass-1.git
   ```
2. **Open index.html**: Simple as that. No build steps required.
3. **API Key**: The current deployment uses a preset API key for demonstration purposes.

## 📐 Design Philosophy

Skyglass was built with a focus on "Atmospheric Intelligence"—the idea that weather data shouldn't just be informative, but visually stunning. Using a combination of radial gradients, CSS animations, and Syne typography, the app transforms raw data into a beautiful, interactive experience.
